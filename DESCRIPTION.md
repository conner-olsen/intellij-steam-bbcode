<p>Steam BBCode language support for IntelliJ-based IDEs.</p>
<p>This plugin is maintained as a separate Steam-focused fork of the original BBCode plugin.</p>
<p>Plugin id: <code>io.github.conner-olsen.steambbcode</code></p>
<p>
  The default registered file name pattern is <code>*.bbcode</code> and can be configured in
  <code>Settings -&gt; Editor -&gt; File Types</code>.
</p>

<br/>

<p>
  Supports Steam-style BBCode features including <code>[olist]...[/olist]</code> and nested list containers.
</p>
